接続設定

Supabaseを設定すると、複数端末・スタッフ間でデータと書類を共有できます。未設定でもこの端末内（ブラウザ）でそのまま使えます。

初期セットアップ（Supabase側で1回だけ実行）

① Supabaseで新規プロジェクト作成 → ② SQL Editorに上記を貼って実行 → ③ Storageで「catalogs」バケットを作成しPublicに設定 → ④ 上のURL / anon keyを入力して保存。

CSV 取込 / 書出

100製剤をまとめて登録するなら、テンプレートに入力 → 取込が一番早いです。
