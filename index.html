貨物照合 × まとめ請求管理
プレビュー環境のため保存できません。GitHub Pages へ配置するか、ファイルを保存してブラウザで開くと自動保存されます。
貨物を選択
左から選ぶか、「＋ 新規 貨物」で届いた貨物を登録します。
請求を選択
まとめ請求書が届いたら「＋ 新規 請求」で作成し、対象の貨物を紐づけます。

明細を貼り付けて取込

インボイスから 品名 と 発注数 を貼り付けます。1行1品、区切りはタブ・カンマどちらでも可。
例：　Rejuran    50

AI読み取り設定

書類をAIで読み取るには Anthropic APIキー が必要です。キーはこの端末のブラウザ内だけに保存され、バックアップには含まれません。（Claude.ai内でこのツールを開いている場合はキー無しで動くことがあります）

公開リポジトリや GitHub Pages にキーを直書きしないでください。本番運用ではキーをサーバ側（Supabase Edge Function 等）に置く構成へ移行します。
